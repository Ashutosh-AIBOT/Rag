## Coding Rules & Implementation Standards

*   **Imports Order:** Group imports by Standard Library, Third-party Libraries (e.g. FastAPI, LangChain), and Local Modules.
*   **Logging:** Every module gets a logger via `logger = get_logger(__name__)`. Log execution entry/exit, timing, and errors.
*   **Synchronous CPU Calls:** Any CPU-bound calls (BM25 tokenization, Cross-Encoder inference) **must** be wrapped in `asyncio.to_thread` or `run_in_threadpool`. Never run them blocking on the main event thread.
*   **Type Safety:** Provide Python type-hints on all function arguments and returns. Validate router inputs/outputs using Pydantic models.
